## 关键指标计算示例（以招商银行为例）

### 输入数据
- 当前股价：35.50元
- 最新分红（2024）：1.48元/股
- 每股收益（EPS）：5.20元

### 计算过程
1. **股息率** = 1.48 ÷ 35.50 × 100% = **4.17%**
2. **分红率** = 1.48 ÷ 5.20 × 100% = **28.46%**
3. **分红增长率** = (1.48 - 1.40) ÷ 1.40 × 100% = **5.71%**

### 指标解读
- ✅ **股息率4.17%**：高于银行理财，具备吸引力
- ✅ **分红率28.46%**：健康范围（<50%），可持续性强  
- ✅ **增长率5.71%**：持续增长，公司盈利向好

## 注意事项

### 数据质量
1. **数据源可靠性**：不同数据源可能有差异
2. **数据延迟**：免费API通常有15分钟延迟
3. **数据完整性**：确保获取完整的历史数据

### 数据验证
```python
def validate_data(self, stock_data):
    """验证数据有效性"""
    issues = []
    
    if stock_data.get('stock_price', 0) <= 0:
        issues.append("股价异常")
    
    if stock_data.get('dividends', {}):
        div_values = list(stock_data['dividends'].values())
        if any(v <= 0 for v in div_values):
            issues.append("分红数据异常")
    
    return issues
```

### 备用方案
建议同时使用多个数据源，进行交叉验证：
```python
def get_data_with_validation(self, stock_code):
    """从多个数据源获取并验证数据"""
    sources = ['yfinance', 'tushare', 'akshare']
    all_data = []
    
    for source in sources:
        try:
            data = self.fetch_from_source(source, stock_code)
            if data:
                all_data.append(data)
        except:
            continue
    
    # 数据一致性检查
    validated_data = self.validate_and_merge(all_data)
    return validated_data
```
